# 新型武器说明

本模组新增创造模式物品栏“新型武器”，用于存放当前和后续设计的枪械武器。

## 基本信息

- Minecraft：1.21.3
- Fabric Loader：0.18.4
- 模组 ID：`guns`
- 创造物品栏：`新型武器`
- 当前武器：
  - `guns:sniper_rifle`：狙击枪
  - `guns:shotgun`：霰弹枪
  - `guns:grenade_launcher`：榴弹枪

## 获取命令

```mcfunction
/give @p guns:sniper_rifle
/give @p guns:shotgun
/give @p guns:grenade_launcher
```

## 狙击枪

狙击枪适合远距离单点精确射击。

- 基础伤害：10
- 最大射程：128 格
- 右键切换狙击镜倍率：`1x -> 2x -> 4x -> 8x -> 16x -> 关闭`
- 开启狙击镜后，左键射击
- 未开启狙击镜时无法射击
- 射击不会穿透第一个被命中的目标
- 开镜时隐藏原版准心，只保留狙击镜准心
- 开镜射击和切换倍率时不会播放主手挥动攻击动画
- 支持与钻石剑同类的附魔书类型，伤害附魔会影响射击伤害

## 霰弹枪

霰弹枪适合近距离扇形范围压制，可同时命中多个目标。

- 最大射程：8 格
- 射击范围：150 度扇形
- 近距离基础伤害：15
- 最远距离最低伤害：3
- 伤害随距离等比衰减
- 击退随距离等比衰减
- 1 格内击退力度约为 5 格，8 格处基本无击退
- 隔墙不会命中目标
- 左键射击，不播放原版主手攻击挥动动画
- 射击效果以烟雾弹道和命中烟雾为主

## 榴弹枪

榴弹枪发射真实弹体，适合中距离范围爆炸打击。

- 左键发射榴弹
- 弹体使用黑色圆形雪球素材
- 弹体按物理抛物线飞行
- 弹体触碰第一个实体或方块时爆炸
- 爆炸强度：4.0，与 TNT 一致
- 爆炸类型：`World.ExplosionSourceType.TNT`
- 爆炸不产生火焰
- 弹体飞行时有烟雾拖尾，用于显示弹道
- 左键射击不播放原版主手攻击挥动动画

## 持枪姿势

三把武器都使用双手持枪姿势。玩家持有狙击枪开镜、持有霰弹枪或持有榴弹枪时，会调整本地玩家双臂姿态，使角色表现为双手持枪。

## 当前限制

- 当前武器均不支持合成获得。
- 榴弹枪弹体会造成真实 TNT 强度爆炸，可能破坏方块。
- 双手持枪姿势主要覆盖本地玩家渲染表现。
