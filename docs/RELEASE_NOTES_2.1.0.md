# Guns v2.1.0 — Survival Loop Update / 生存循环更新

## English

Guns v2.1.0 turns the mod into a complete vanilla-survival weapons experience for Minecraft 1.21.3 Fabric.

- Craft six weapons with vanilla materials.
- Manufacture five ammunition types. Sniper Rifle and SMG share Rifle Round.
- Repair weapons with iron ingots in an Anvil.
- Install Precision Barrel, Cooling System, and Reinforced Receiver in a Smithing Table.
- Play in English or Simplified Chinese through Minecraft's native language menu.
- Run on multiplayer and dedicated servers with server-authoritative gameplay validation.

### Requirements

- Minecraft 1.21.3
- Fabric Loader 0.18.4 or newer
- Fabric API 0.114.1+1.21.3 or newer
- Java 21

Download the non-sources JAR for normal play. The release also contains a sources JAR for reference.

## 简体中文

Guns v2.1.0 将模组完善为适用于 Minecraft 1.21.3 Fabric 的原版生存枪械体验。

- 使用原版材料合成六把枪械。
- 制造五种弹药；狙击枪与冲锋枪共用步枪弹。
- 在铁砧中使用铁锭维修枪械。
- 在 Smithing Table 安装精密枪管、冷却系统和强化枪机。
- 使用 Minecraft 原生语言菜单在简体中文与英文之间切换。
- 支持多人和 Dedicated Server，关键玩法由服务端权威验证。

### 运行要求

- Minecraft 1.21.3
- Fabric Loader 0.18.4 或更高版本
- Fabric API 0.114.1+1.21.3 或更高版本
- Java 21

正常游玩请下载非 sources 的 JAR；Release 同时提供 sources JAR 供查阅。
